# Repository status

This repository is a public support repository for submitted manuscript **NXSUST-D-26-01414** in *Next Sustainability*.

## Current status

- Public GitHub support repository is available.
- Processed CSV tables, figure data, figures, and notebooks are included.
- The repository is intended for review support and reproducibility checking.
- A Zenodo DOI has not yet been minted for this initial upload and can be added during revision if requested.

## Claim-control note

The workflow is a decision-support and prioritization framework. It does not experimentally validate candidate cathodes, does not claim discovery of new cathode materials, and does not treat text-mined evidence as confirmation of electrochemical performance.
