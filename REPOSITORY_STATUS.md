This repository is the public code/data-support repository for the submitted *Next Sustainability* manuscript NXSUST-D-26-01414.

## Current status

- Public GitHub repository available.
- Processed data, figure data, supplementary tables, figures, audit files, notebooks, environment files, and licenses are included.
- No Zenodo DOI has been minted for this initial public upload.

## Scope statement

The repository supports a criticality-aware decision-support workflow. It does not provide experimental validation and does not claim discovery of new cathode materials.
