# Table captions

## Table 1. Dataset and candidate filtering funnel.
Summary of dataset reduction from Materials Project sodium-ion electrode records to final candidate subsets. Counts report the sequential effect of basic electrochemical screening, hard-exclusion filtering, conservative earth-abundance proxy filtering, exact ChemDataExtractor evidence matching, and final deduplication.

## Table 2. Leakage-corrected grouped cross-validation model performance.
Best grouped-CV model performance by target and descriptor protocol after Notebook 06 leakage auditing. The preferred manuscript values use target-specific leakage-corrected variants: Na-content near-target features are removed for capacity and energy, and stability_charge/stability_discharge are removed for stability_worst. Full-feature reconstruction metrics are retained in Supplementary Table S4.

## Table 3. Family-level screening and evidence summary.
Family-level counts for all records, basic screening passes, hard-exclusion-free candidates, conservative candidates, and exact text-mined evidence matches. The CDE evidence count is computed independently from screening-pass status and should be read as a text-mined literature-evidence signal, not validation.

## Table 4. Top-20 candidate shortlist.
Top-ranked entries from the deduplicated manuscript shortlist. The score ranks candidates for follow-up prioritization under the current workflow and should not be interpreted as experimental validation or guaranteed sustainability. Full 95-entry shortlist is provided in Supplementary Table S6.
